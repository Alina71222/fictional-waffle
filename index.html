Сегодня я расскажу, из каких частей состоит доменное имя моего сайта и на каких уровнях иерархии они находятся.

Мой сайт размещён на GitHub Pages, и его адрес: alinabelaykova.github.io.

Это доменное имя состоит из следующих уровней:

  • Корневой домен — неявная точка в конце («.»), технический уровень DNS.
  • Домен верхнего уровня (TLD) — .io, изначально предназначался для Британских территориальных вод, но сейчас часто используется для технологических проектов.
  • Домен второго уровня — alinabelaykova, это моё имя пользователя на GitHub, зарегистрированное как часть GitHub Pages.
  • Поддомен — в данном случае отсутствует, так как используется «голый» домен alinabelaykova.github.io. Если бы был адрес вида blog.alinabelaykova.github.io, то blog был бы поддоменом.

Таким образом, полное доменное имя: alinabelaykova.github.io — это комбинация имени пользователя и домена GitHub (github.io), предоставленного платформой для бесплатного хостинга.
